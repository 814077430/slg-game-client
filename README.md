# SLG MUD Client

SLG 游戏 MUD（文字冒险）客户端 - 基于文本的交互式游戏客户端

## 🎮 特性

- ✅ 1024x1024 大世界支持
- ✅ 中心皇城 + 四大州 + 蛮荒带
- ✅ 纯文字界面
- ✅ 小地图显示
- ✅ 命令别名系统
- ✅ 命令历史记录
- ✅ 实时输出格式化
- ✅ 支持 TCP 长连接
- ✅ 与 slg-game 服务器完全兼容

## 🚀 快速开始

### 编译

```bash
# MUD 客户端
cd cmd/mud
go build -o mud-client .

# 普通客户端
cd cmd
go build -o slg-client .
```

### 运行

```bash
# 连接本地服务器
./mud-client

# 连接远程服务器
./mud-client server:port
```

## 📖 游戏命令

### 移动命令

| 命令 | 别名 | 说明 |
|------|------|------|
| `go north` | `n` | 向北移动 |
| `go south` | `s` | 向南移动 |
| `go east` | `e` | 向东移动 |
| `go west` | `w` | 向西移动 |
| `go northeast` | `ne` | 向东北 |
| `go northwest` | `nw` | 向西北 |
| `go southeast` | `se` | 向东南 |
| `go southwest` | `sw` | 向西南 |

### 基础命令

| 命令 | 别名 | 说明 |
|------|------|------|
| `login <用户名> <密码>` | - | 登录 |
| `register <用户名> <密码>` | - | 注册 |
| `look` | `l` | 查看周围 + 小地图 |
| `status` | `st` | 查看状态 |
| `inventory` | `i` | 查看背包 |
| `build <建筑>` | - | 建造建筑 |
| `work` | - | 工作 |
| `rest` | - | 休息 |
| `say <消息>` | - | 说话 |
| `who` | - | 在线玩家 |
| `map` | - | 显示小地图 |
| `info` | - | 区域信息 |
| `help` | - | 帮助 |

### 特殊命令（以/开头）

| 命令 | 说明 |
|------|------|
| `/quit` | 退出游戏 |
| `/clear` | 清屏 |
| `/refresh` | 刷新界面 |
| `/history` | 命令历史 |
| `/alias` | 别名列表 |
| `/help` | 帮助 |
| `/map` | 小地图 |

## 🎯 游戏流程

```
1. 启动客户端
   $ ./mud-client

2. 注册账号
   > register player1 password123

3. 登录
   > login player1 password123

4. 查看周围和小地图
   > look

5. 移动探索
   > n (向北)
   > e (向东)
   > go northeast (向东北)

6. 查看状态
   > status

7. 建造建筑
   > build farm

8. 工作获取资源
   > work

9. 查看区域信息
   > info

10. 显示帮助
    > help
```

## 🔧 自定义别名

在 `handler.go` 中添加：

```go
h.RegisterAlias("缩写", "完整命令", "描述")
```

## 📊 输出格式

客户端自动格式化服务器输出：

- ✅ 成功消息
- ❌ 失败消息
- 🎮 欢迎消息
- 📍 位置信息
- 💰 资源信息
- 👤 玩家信息

## 🛠️ 开发

### 添加新命令

1. 在 `handler.go` 的 `processGameCommand` 中添加命令解析
2. 在 `client.go` 中添加对应的发送方法
3. 在 `ui.go` 中添加界面显示方法
4. 更新帮助文档

### 添加新别名

```go
h.RegisterAlias("x", "examine", "检查物品")
```

### 自定义界面

修改 `ui.go` 中的界面显示方法：
- `ShowHeader()` - 头部
- `ShowPlayerInfo()` - 玩家信息
- `ShowMiniMap()` - 小地图
- `ShowAreaInfo()` - 区域信息
